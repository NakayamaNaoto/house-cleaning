# Target Site Notes

## レイアウト概要
- 固定ヘッダー: ロゴ、グローバルナビ（トップ/メニュー/ギャラリー/FAQ ほか）、LINE・Instagram アイコン、問い合わせボタン
- ヒーロー: キャッチコピー（例: 安さで業者を選んでませんか？）とリード文＋全面背景画像
- 説明セクション: 「お掃除代行の頼み先に困っている方に！」などのメッセージを大きく配置
- 実績/特徴セクション: 「55リフォーム名古屋店のここがすごい！」見出し配下に縦並びのカード（画像 + h3 + テキスト）×3
- 理由セクション: 「『頼んで良かった！』の声...」見出し、POINT 01?03 のカード（背景画像 + 番号バッジ + 説明）
- CTA バー: 「お急ぎの場合は電話...」「052-990-3015」「営業時間 8:00〜22:00」
- 料金表: 「お掃除代行セットプラン料金」見出し、番号付きカード 3 枚（画像・タイトル・本文）
- 追加グリッド: data-lib 20762 部分でカード 6 枚（画像 + 見出し + テキスト）
- フッター: ロゴ/社名/住所/電話、主要リンク 2列（トップページ/メニュー.../プライバシー等）

## カラーパレット & タイポグラフィ（target-site.css より）
- ベース背景 `#f7f7f7`
- ベーステキスト `#131313`
- アクセント `#393939`
- 補助カラー `#828080`, ボタン `#a1a1a1`
- フォント: すべて Shippori Mincho（タイトル/本文/英字）
- コンテナ幅: 1170px（`[data-w-width="1"]`）
- レスポンシブ: 768px 以下でフォントサイズ縮小、`max-width:94%`

## コンテンツの主な文言
1. ヒーロー: 「安さで業者を選んでませんか？」「適正価格で安心なお掃除代行を。」など
2. アピール見出し: 「お掃除代行 55リフォーム名古屋店のここがすごい！」サブ「キッチンや水回り...」
3. カード内容:
   - 新品同様の美しさを目指すクオリティ
   - 2段階無料見積もり（LINE → 現地）
   - 依頼確定まで完全無料（現地調査推奨）
4. POINT セクション:
   - 01 水回り汚れを根本から除去
   - 02 人柄の良いスタッフ
   - 03 各所専門のプロが対応
5. CTA: 電話番号 052-990-3015／営業時間 8:00〜22:00
6. 料金プランカード（例: お掃除代行3点セット、水回り5点セット、キッチン＆レンジフードセット）

## 実装上のメモ
- 多くのセクションが左右 2カラム（テキスト+画像）またはカードグリッド
- 背景は淡いグレー + 白カード、タイトルはアクセント色 (#828080/#393939) と Mincho フォント
- 各カード h3 と本文 p が複数行
- 追加カードグリッド(6枚)は `flex_list3` で 3列レイアウト
- 主要CTA（LINE/問い合わせボタン/電話）を複数配置する必要あり
